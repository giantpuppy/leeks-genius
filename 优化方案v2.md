# 排期助手 - 甘特图产品逻辑调整方案

## 产品定位重定义

甘特图 = 私人定制排期图。用户关注的核心信息：
- **演出时间**（日期 + 时间）
- **演出地点**（剧场）
- **卡司**（演员阵容）
- **剧目名称**

**不要**：座位、票价（这些信息放在详情里即可，甘特图不展示）

**核心动作**：对场次标记状态 -> 状态同步到月历

---

## 阶段一：数据模型改造

### 1. Performance 增加 status 字段

```dart
enum PerformanceStatus {
  unmarked,    // 未标记（默认）
  wantToSee,   // 想看
  bought,      // 已买
}

class Performance {
  int? id;
  int showId;
  String date;
  String? time;
  String? seat;       // 保留字段，但甘特图不显示
  double? price;      // 保留字段，但甘特图不显示
  String? status;     // 'unmarked' | 'want_to_see' | 'bought'
  String? createdAt;
  
  // toMap / fromMap / copyWith 同步更新
}
```

### 2. 数据库表改造

- performances 表增加 `status TEXT DEFAULT 'unmarked'`
- IO 端（database_helper_io.dart）：升级数据库 version 到 2，onUpgrade 添加列
- Web 端（database_helper_web.dart）：同步表结构

### 3. Web 内存 DB 兼容

- _WebDB 的 _load / _save 支持 status 字段（JSON 天然支持，无需改动）

## 阶段二：甘特图界面改造

### 视觉简化

| 现在 | 改造后 |
|------|--------|
| 甘特条显示时间 | 甘特条显示时间 + 状态图标 |
| 所有场次都显示 | 未标记的场次淡化显示（半透明） |
| 左侧有座位/票价 | 移除 |

### 状态颜色

| 状态 | 甘特条颜色 | 标记 |
|------|-----------|------|
| 未标记 | 灰色 #9CA3AF | 无 |
| 想看 | 紫色 #811FE2 | 空心星星 |
| 已买 | 绿色 #34D399 | 实心对勾 |

### 交互改造

**点击甘特条** -> 底部面板：
- 显示：剧目名、剧场、日期时间、卡司
- 操作按钮组：
  - 【想看】紫色空心按钮 -> 标记为想看
  - 【已买】绿色实心按钮 -> 标记为已买
  - 【取消标记】灰色文字按钮 -> 恢复未标记
  - 【编辑】修改日期/时间/卡司
  - 【删除】删除场次

**长按/右键甘特条** -> 快捷菜单：
- 标记想看
- 标记已买
- 删除

**左侧快速添加（+按钮）** -> 简化表单：
- 日期（必填）
- 时间（默认四种选择，19:30，14:30,19:00,14:00）
- 卡司（可选，快速输入）

## 阶段三：月历联动

### 月历显示规则

- **只显示已标记**（想看/已买）的场次
- 未标记的场次不显示在月历上（避免甘特图和月历重复混乱）

### 月历标记点颜色

| 状态 | 月历标记点 |
|------|-----------|
| 想看 | 紫色圆点 |
| 已买 | 绿色圆点 |
| 同一天两种状态 | 两个并列小点 |

### 月历底部列表

点击日期 -> 底部列表显示当天标记场次：
- 左侧：状态图标（想看=空心星，已买=实心勾）
- 中间：剧目名 + 时间 + 剧场
- 右侧：点击可跳转详情/修改状态

### 月历筛选

月历页面顶部增加筛选栏：
- 全部 / 想看 / 已买

---

## 阶段四：主页/仪表盘改造

### 统计卡片

| 指标 | 含义 |
|------|------|
| 想看 | 标记想看的场次数量 |
| 已买 | 已买票的场次数量 |
| 即将演出 | 未来 7 天内已买的场次 |

### 甘特图筛选

甘特图工具栏增加筛选：
- 全部 / 想看 / 已买

---

## 验收路径

1. 在甘特图添加一个剧目场次 -> 只填日期、时间、剧场、卡司
2. 点击甘特条 -> 底部面板显示【想看】【已买】按钮
3. 点击【想看】-> 甘特条变紫色，切到月历 -> 该日期出现紫色标记
4. 点击【已买】-> 甘特条变绿色，月历标记变绿色
5. 刷新页面 -> 所有状态保留
6. 月历筛选"已买" -> 只显示已买场次
7. 主页统计卡片数字正确
